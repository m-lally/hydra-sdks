Webhook Management

To set up webhooks for real-time notifications:

  1. Go to your Shopify Admin → Settings → Notifications
  2. Scroll down to "Webhooks"
  3. Click "Create webhook"
  4. Choose an event (e.g., Order creation)
  5. Set the format to JSON
  6. Enter the webhook URL: https://your-app-url/webhooks/orders/create
  7. Save the webhook

Note: Replace your-app-url with your actual app URL.
